## 커뮤니케이션

우리는 디자이너와 개발자를 더 가까이 모으려는 모든 노력이 가치 있음을 믿습니다. 사용하는 언어는 항상 정확해야 하지만 명확성을 희생해서도 안 됩니다.

* 디자이너와 엔지니어 모두에게 명확한 의미를 가진 어휘 사용하기
* 상호 이해를 돕는 단어를 선택하고, 특정한 용어에만 의존하지 않기
* 달성하려는 바와 달성하지 않으려는 바를 명확히 전달하기
* 새로운 단어에 대한 예시와 설명 제공하기
* 글쓰기와 말하기에서 어휘 사용의 일관성을 유지하기
