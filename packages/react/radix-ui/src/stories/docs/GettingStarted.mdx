# 시작하기

## DS 디자인 시스템 소개

**DSDS Radix-UI** 는 크게 **두 가지 주요 개념**으로 구성되어 있습니다.

- 모듈성과 접근성에 중점을 둔 **React 컴포넌트** 세트
- **TailwindCSS** 기반 테마/스타일링 시스템으로, 애플리케이션의 테마 설정 및 사용자 정의 컴포넌트 스타일링 가능
